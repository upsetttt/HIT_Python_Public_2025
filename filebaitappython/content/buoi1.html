Bài 1:

Python là ngôn ngữ thông dịch.

✅ Giải thích:

- Python không biên dịch trực tiếp mã nguồn sang mã máy như ngôn ngữ C, C++, Java,... mà mã nguồn .py được dịch sang mã bytecode (file .pyc) và sau đó được thực thi bởi một trình thông dịch (interpreter) – cụ thể là Python Virtual Machine (PVM).

🔁 Quy trình hoạt động:

  • Bước 1: Viết mã nguồn Python (.py)
  • Bước 2: Trình thông dịch Python biên dịch mã nguồn sang bytecode (.pyc)
  • Bước 3: Bytecode được thực thi bởi Python Virtual Machine (PVM)

    • 🔍 Ví dụ:

    print("Hello, world!")

    ✅ Kết luận:

    Python là ngôn ngữ thông dịch vì mã nguồn được biên dịch sang bytecode và thực thi bởi trình thông dịch, không phải biên dịch trực tiếp sang mã máy.
